# _**Create a new push**_

### _**Steps**_

- _**Set Global Git Configuration**_

  ```yml
  git config --global user.name "Your name"

  # Email must be in this format
  git config --global user.email "your-github-username@users.noreply.github.com"
  ```

- _**Add README.md**_

  ```yml
  git add README.md
  ```

- _**Add data.json**_

  ```yml
  git add data.json
  ```

- _**Push**_

  ```yml
  git push
  ```
